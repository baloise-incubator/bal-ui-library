## Basic

The grid is based on 12 columns. On mobile the columns will use the full width.

<Code
  language="html"
  code={`
<div class="columns">
    <div class="column">
      <div class="has-background-green-3 p-small has-radius-normal">First column</div>
    </div>
    <div class="column">
      <div class="has-background-green-3 p-small has-radius-normal">Second column</div>
    </div>
    <div class="column">
      <div class="has-background-green-3 p-small has-radius-normal">Third column</div>
    </div>
    <div class="column">
      <div class="has-background-green-3 p-small has-radius-normal">Fourth column</div>
    </div>
  </div>
`}
/>

## Rows

By adding the CSS class `is-multiline` the columns can be shown on more than one row.

<Code
  language="html"
  code={`
<div class="columns is-multiline">
  <div class="column is-full">
    <div class="has-background-green-3 p-small has-radius-normal">First row</div>
  </div>
  <div class="column is-full">
    <div class="has-background-red-3 p-small has-radius-normal">Second row</div>
  </div>
</div>
`}
/>

## Nested

Grids can be nested to have more flexibility in your structure.

Just keep the structure columns > column > columns > column > ...

<Code
  language="html"
  code={`
<div class="columns">
  <div class="column is-half">
    <div class="has-background-green-3 p-small has-radius-normal">Half</div>
  </div>
  <div class="column is-half">
    <div class="columns">
      <div class="column is-half">
        <div class="has-background-red-3 p-small has-radius-normal">Half</div>
      </div>
      <div class="column is-half">
        <div class="has-background-red-3 p-small has-radius-normal">Half</div>
      </div>
    </div>
  </div>
</div>
`}
/>

## Space

By default add a gap `var(--bal-column-gap)` of 1rem between the columns.
This can be avoided with setting the CSS class `is-gapless`.

<Code
  language="html"
  code={`
<div class="columns is-gapless">
    <div class="column">
      <div class="has-background-green-3 p-small has-radius-normal">First column</div>
    </div>
    <div class="column">
      <div class="has-background-green-3 p-small has-radius-normal">Second column</div>
    </div>
    <div class="column">
      <div class="has-background-green-3 p-small has-radius-normal">Third column</div>
    </div>
    <div class="column">
      <div class="has-background-green-3 p-small has-radius-normal">Fourth column</div>
    </div>
  </div>
`}
/>

## Breakpoint

The columns can be responsive and have different sizes on the breakpoints.

<Code
  language="html"
  code={`
<div class="columns is-mobile">
    <div class="column is-three-quarters-mobile is-two-thirds-tablet is-half-desktop is-one-third-widescreen is-one-quarter-fullhd">
      <div class="has-background-green-3 p-small has-radius-normal">
        is-three-quarters-mobile<br>
        is-two-thirds-tablet<br>
        is-half-desktop<br>
        is-one-third-widescreen<br>
        is-one-quarter-fullhd
      </div>
    </div>
    <div class="column">
      <div class="has-background-green-3 p-small has-radius-normal">auto</div>
    </div>
    <div class="column">
      <div class="has-background-green-3 p-small has-radius-normal">auto</div>
    </div>
    <div class="column">
      <div class="has-background-green-3 p-small has-radius-normal">auto</div>
    </div>
  </div>
`}
/>

## Vertical Alignment

To center the columns of the grid vertically set `is-vcentered` CSS class.

<Code
  language="html"
  code={`
<div class="columns is-vcentered">
    <div class="column is-two-thirds">
      <div class="has-background-green-2 p-small has-radius-normal">First column</div>
    </div>
    <div class="column">
      <div class="has-background-green-2 p-small has-radius-normal">Second column with some random large text to make a good case here. Unfortunately, more content is needed to make it visible that the first column is now vertically centered.</div>
    </div>
  </div>
`}
/>

## Horizontal Alignment

To center the columns of the grid set `is-centered` CSS class.

<Code
  language="html"
  code={`
<div class="columns is-centered">
    <div class="column is-one-third">
      <div class="has-background-green-2 p-small has-radius-normal">My centered column</div>
    </div>
  </div>
`}
/>

## Stretch

To stretch the inner element of a column use the CSS class `is-fullheight`.

<Code
  language="html"
  code={`
<div class="columns is-multiline">
    <div class="column is-two-thirds">
      <div class="has-background-green-2 p-small has-radius-normal is-fullheight">Stretches to the full height of the row</div>
    </div>
    <div class="column is-one-third">
      <div class="has-background-green-2 p-small has-radius-normal is-fullheight">Second column with some random large text to make a good case here. Unfortunately, more content is needed to make it visible that the first column is now vertically centered.</div>
    </div>
    <div class="column is-half">
      <div class="has-background-green-2 p-small has-radius-normal is-fullheight">Second column with some random large text to make a good case here. Unfortunately, more content is needed to make it visible that the first column is now vertically centered.</div>
    </div>
    <div class="column is-half">
      <div class="has-background-green-2 p-small has-radius-normal is-fullheight">Stretches to the full height of the row</div>
    </div>
  </div>
`}
/>
