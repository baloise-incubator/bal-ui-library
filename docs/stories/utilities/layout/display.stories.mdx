<Lead>Specifies the display behavior of an element on a page.</Lead>

## Classes

<CssLayoutDisplay />

## Hidden

An element that is hidden is not visible and does not occupy any space.

<Code
  language="html"
  code={`
<div class="is-flex fg-normal">
  <div class="is-hidden has-background-green p-normal has-radius-normal">1</div>
  <div class="has-background-green p-normal has-radius-normal">2</div>
  <div class="has-background-green p-normal has-radius-normal">3</div>
</div>
`}
/>

## Block

A block element begins on a new line.

<Code
  language="html"
  code={`
<div class="is-block has-background-green has-text-centered p-normal has-radius-normal mb-normal">1</div>
<div class="is-block has-background-green has-text-centered p-normal has-radius-normal mb-normal">2</div>
<div class="is-block has-background-green has-text-centered p-normal has-radius-normal mb-normal">3</div>
`}
/>

## Inline

An inline element does not initiate on a new line and takes up width as required.

<Code
  language="html"
  code={`
<div class="is-inline has-background-green has-text-centered p-normal has-radius-normal mx-normal">1</div>
<div class="is-inline has-background-green has-text-centered p-normal has-radius-normal">2</div>
<div class="is-inline has-background-green has-text-centered p-normal has-radius-normal mx-normal">3</div>
`}
/>

## Inline Block

Inline block behaves much like an inline element, but it also respects properties such as width, height, and top/bottom paddings/margins.

<Code
  language="html"
  code={`
<div class="is-inline-block has-background-green has-text-centered p-normal has-radius-normal">1</div>
<div class="is-inline-block has-background-green has-text-centered p-normal has-radius-normal mx-normal">2</div>
<div class="is-inline-block has-background-green has-text-centered p-normal has-radius-normal">3</div>
`}
/>

## Flex

Sets the element to display as a block-level flex container.

<Code
  language="html"
  code={`
<div class="is-flex fg-normal">
    <div class="is-flex-grow-1 has-background-green has-text-centered p-normal has-radius-normal">1</div>
    <div class="is-flex-grow-1 has-background-green has-text-centered p-normal has-radius-normal">2</div>
    <div class="is-flex-grow-1 has-background-green has-text-centered p-normal has-radius-normal">3</div>
</div>
`}
/>

## Inline Flex

Sets the element to display as an inline-level flex container.

<Code
  language="html"
  code={`
<div class="is-inline-flex fg-normal">
    <div class="is-flex-grow-1 has-background-green has-text-centered p-normal has-radius-normal">1</div>
    <div class="is-flex-grow-1 has-background-green has-text-centered p-normal has-radius-normal">2</div>
    <div class="is-flex-grow-1 has-background-green has-text-centered p-normal has-radius-normal">3</div>
</div>
`}
/>

## Responsive

Tailored responsive options are at your disposal for customizations according to screen size.
Simply append a responsive breakpoint keyword with a dash, like `is-hidden-tablet`, to apply the corresponding responsive class.

<Code
  language="html"
  code={`
<div class="has-background-yellow-2 p-small mb-small is-hidden-mobile">is-hidden-mobile</div>
<div class="has-background-yellow-3 p-small mb-small is-hidden-tablet">is-hidden-tablet</div>
<div class="has-background-yellow-4 p-small mb-small is-hidden-touch">is-hidden-touch</div>
<div class="has-background-red-2 p-small mb-small is-hidden-desktop">is-hidden-desktop</div>
<div class="has-background-red-3 p-small mb-small is-hidden-high-definition">is-hidden-high-definition</div>
<div class="has-background-red-4 p-small mb-small is-hidden-widescreen">is-hidden-widescreen</div>
<div class="has-background-red-5 p-small mb-small is-hidden-fullhd">is-hidden-fullhd</div>
<hr>
<div class="has-background-purple-2 p-small mb-small is-hidden-tablet-only">is-hidden-tablet-only</div>
<div class="has-background-purple-3 p-small mb-small is-hidden-desktop-only">is-hidden-desktop-only</div>
<div class="has-background-purple-4 p-small mb-small is-hidden-high-definition-only">is-high-definition-only</div>
<div class="has-background-purple-5 p-small mb-small is-hidden-widescreen-only">is-hidden-widescreen-only</div>
`}
/>
