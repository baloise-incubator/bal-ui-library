## Classes

<CssTypographyWeight />

## Bold

Both font families offer a bold style option.

<Code language="html" code={`
<h1 class="has-text-weight-bold title">Bold - Title</h1>
<p class="has-text-weight-bold">Bold - Lorem ipsum dolor sit, amet consectetur adipisicing elit.</p>
`} />

## Regular

Onyl the **text** font families, which is default, offer a regular style option.

<Code language="html" code={`
<p>Regular - Lorem ipsum dolor sit, amet consectetur adipisicing elit.</p>
`} />

## Light

Onyl the **title** font families offer a light style option.

<Code language="html" code={`
<h1 class="has-text-weight-light title">Light - Title</h1>
`} />
