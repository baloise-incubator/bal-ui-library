## Installation

Introducing CSS utility classes meticulously crafted around our [design tokens](?path=/docs/foundation-design-tokens--documentation),
seamlessly integrated into the `@baloise/design-system-css` package. These classes are accessible in
both **CSS** and **SCSS** formats, ensuring versatile usage.

The `@baloise/design-system-css` is included in the component package, so if that is already installed skip the next step.

```bash
npm add @baloise/design-system-css
```

The CSS utility package is designed in a modular fashion, allowing for selective imports based on the specific needs of the application.

```css
// SASS mixins and variables
@import '@baloise/design-system-css/sass/mixins';

// Resets CSS for all browser
@import '@baloise/design-system-css/css/normalize';
@import '@baloise/design-system-css/css/structure';

// Custom font faces
@import '@baloise/design-system-css/sass/font';

// Core CSS, always required
@import '@baloise/design-system-css/css/core';

// CSS utilities classes (optional)
@import '@baloise/design-system-css/css/border';
@import '@baloise/design-system-css/css/color';
@import '@baloise/design-system-css/css/display';
@import '@baloise/design-system-css/css/flex';
@import '@baloise/design-system-css/css/grid';
@import '@baloise/design-system-css/css/opacity';
@import '@baloise/design-system-css/css/radius';
@import '@baloise/design-system-css/css/shadow';
@import '@baloise/design-system-css/css/spacing';
@import '@baloise/design-system-css/css/typography';
```

## Component styling

Import `@baloise/design-system-css/sass/mixins` into your component's style imports to gain access to a comprehensive collection of design tokens and Sass mixins.

<ScssSource code={`// SASS mixins and variables
@import '@baloise/design-system-css/sass/mixins';

.my-component {
${' '}${' '}background: var(--bal-color-green);
}

@include desktop() {
${' '}${' '}// add your custom styles here
${' '}${' '}.my-component {
${' '}${' '}${' '}${' '}background: var(--bal-color-red);
${' '}${' '}}
}`} />
