## Color Overview

Our colors are a core part of our design system. We use color to visually communicate information, functions, and personality in our experiences.
We are committed to complying with the Web Content Accessibility Guidelines AA standard contrast ratios.

### Core Color

Blue dictates the visual appearance of Baloise and is present on every site. The Logo, text, buttons and links are blue.

<ColorCards>
  <ColorCard background color="primary" />
</ColorCards>

### Typography Color

The main color for text is the core blue color and for dark backgrounds the inverted core color white.

<ColorCards>
  <ColorCard background color="primary" />
  <ColorCard background color="white" />
</ColorCards>

### Backgrounds

There are eight colors which can be used for backgrounds.
However, the white color represents the default. The other colors should be used only if the context allows.

### Primary colors

Our primary colors play the main role in our brand identity. A vibrant set of fresh shades is the center point of the color palette. They are used to provide accessibility, simplicity, and consistency throughout all brand communications. They are used for Baloise Shapes, Illustrations, background colors and to highlight design elements.

<ColorCardRow color="green" />
<ColorCardRow color="purple" />
<ColorCardRow color="red" />
<ColorCardRow color="yellow" />

### Neutral colors

Neutral colors are typically used for text, borders and subtle backgrounds when we don't want to draw too much attention to a particular design element.

<ColorCardRow color="grey" />

### Functional Colors

These colors are a functional extension of the accent color and are meant to be used only for specific cases and situations.

<ColorCardRow color="blue" />
<ColorCardRow color="light-blue" />

### Alert colors

To draw attention to important information or actions we used our Alert colors palette.

<ColorCardRow color="success" />
<ColorCardRow color="warning" />
<ColorCardRow color="danger" />

## Color Combinations

Color combinations always need to stay in the same color spectrum. For example Purple with Purple Light. There is no color coding within the Baloise appearance. All Baloise primary colors are equal and should be
used as such.
